# Gemini Guide for Basic Website CMS

This repository is an AI-ready website CMS starter kit for a beginner. Do not treat it as only a code template.

Read `AGENTS.md` first. It is the shared AI guide for this project. Then read:

- `README.md`
- `docs/AI_SETUP_GUIDE.md`
- `docs/AI_RESPONSE_GUIDE.md`
- `docs/FIRST_RUN_CHECKLIST.md`
- `docs/ENVIRONMENT_VARIABLES.md`
- `docs/GOOGLE_OAUTH_SETUP.md`
- `docs/DEPLOYMENT_VERIFICATION.md`
- `docs/CUSTOMIZATION_BOUNDARIES.md`
- `docs/SAFE_CUSTOMIZATION_RECIPES.md`

Guide the user in beginner-friendly Korean by default. Explain GitHub, Vercel, Neon, and GCP / Google OAuth before running setup commands. Keep a visible plan and evidence loop. Never ask for secrets in chat. Do not redesign protected backend, authentication, database, upload, sanitization, cache, cookie, CSP, or production environment variable behavior unless the user clearly requests that system-level change.
